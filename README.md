# House Price Prediction

## Project Overview
This project predicts house prices based on various property features using Machine Learning techniques. The model analyzes factors such as area, number of bedrooms, bathrooms, parking spaces, and other housing attributes to estimate the price of a house.

## Features
- House price prediction using Machine Learning
- User-friendly interface
- Real-time prediction
- Data preprocessing and feature engineering
- Model evaluation and performance analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- flask
- HTML

## Dataset Features
- Area
- Main Road Access
- Furnishing Status
- Basement
  
## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Price Prediction

## Results
The model predicts house prices based on user-provided property details and helps users estimate property values efficiently.

## How to Run
```bash
pip install -r requirements.txt
flask run app.py
```
